## 🎯 What We Do

> **Write once. Publish everywhere.**

**PipePub** is an open-source GitHub Actions pipeline that turns your **markdown files into published articles** on all major content distribution platforms — with tags, tables, and image support.

- ✅ **Write in markdown** — no platform-specific formatting
- ✅ **Upload or write directly on GitHub** — no terminal needed
- ✅ **Tables supported** — automatically converted to GitHub Gists
- ✅ **Drafts by default** — review before publishing live
- ✅ **Multiple platforms** — Dev.to, Ghost, Hashnode, Medium, and more coming

<a id="get-started"></a>

## 🚀 Get Started

1. **Use our template** — click "Use this template" on [github.com/pipepub/pipepub](https://github.com/pipepub/pipepub)
2. **Add your API tokens** as GitHub secrets
3. **Write your article** — upload a `.md` file or create one directly on GitHub
4. **Publish** — your article appears on your platforms within seconds

No terminal. No git commands. Just your browser.
